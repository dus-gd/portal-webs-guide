# portal-webs-guide

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放和维护一系列独立的 HTML 页面。这些页面彼此独立，不依赖于特定的域名或外部网站，旨在作为静态页面存档或轻量级信息展示的载体。

仓库本身不提供搜索、导航或动态功能，仅作为页面的集中存储和版本管理场所。

## 目录说明

- `pages/` —— 存放所有独立 HTML 页面文件。
- `assets/` —— 存放页面引用的公共资源（如 CSS、JavaScript、图片等）。
- `docs/` —— 可选，存放与页面内容相关的文档或说明文件。

> 注：目录结构可能根据实际需要调整，请以仓库最新状态为准。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可单独打开或部署。
- 页面内容以信息展示、资料归档或工具类页面为主。
- 页面之间不互相依赖，也不依赖外部网络资源（除非明确标注）。

## 维护说明

- 欢迎提交 Pull Request 增加或更新页面内容。
- 新增页面请放置在 `pages/` 目录下，并确保文件命名清晰、无歧义。
- 请避免在页面中引入外部不可控资源（如第三方 CDN），以保持页面的自包含性。
- 本仓库不保证对所有页面内容的持续更新或兼容性维护。
